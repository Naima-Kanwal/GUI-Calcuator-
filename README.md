GUI Calculator (Qt & C++)

Description:
A simple GUI-based calculator developed in C++ using Qt Creator.
It performs basic arithmetic operations and was created as a Programming Fundamentals Lab semester project (1st Semester).

Features:
Graphical user interface
Addition, subtraction, multiplication, division
Clear button
Easy to use

Tools Used:
C++
Qt Creator
Qt Widgets

How to Run:
Open the project in Qt Creator and click Run.
